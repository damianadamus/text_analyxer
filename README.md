# Simple Text Analyzer

A simple text analyzer written in Python, designed to run in the terminal.

## Features
- Counts words, characters, and lines.
- Counts characters excluding spaces.
- Calculates the average word length.
- Identifies the 5 most frequently occurring words.
- Supports manual input and loading of `.txt` files.

## Instructions
1. Make sure you have Python (3.6+) installed.
2. Download the `text_analyzer.py` file.
3. Run the following in the terminal: python3 text_analyzer.py
